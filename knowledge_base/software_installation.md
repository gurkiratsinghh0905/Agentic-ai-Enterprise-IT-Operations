# Software Installation Guide

## Approved Software

* Microsoft Office
* Teams
* VPN Client
* Endpoint Security Agent
* Adobe Reader

## Installation Process

1. Open Software Center.
2. Select approved application.
3. Click Install.
4. Restart device if prompted.

## Restricted Software

Employees may not install:

* Torrent software
* Unauthorized remote access tools
* Cracked software

## Escalation

Contact IT if installation fails after two attempts.
