# VPN Troubleshooting Guide

## Purpose

This guide helps employees troubleshoot VPN connectivity issues while accessing corporate resources remotely.

## Common VPN Issues

### Issue 1: Invalid Credentials

Symptoms:

* Authentication failed
* Username or password incorrect

Resolution:

1. Verify username format:
   [firstname.lastname@company.com](mailto:firstname.lastname@company.com)
2. Reset password if expired.
3. Ensure Caps Lock is disabled.
4. Retry login.

### Issue 2: VPN Client Not Launching

Symptoms:

* VPN application crashes
* Application does not open

Resolution:

1. Restart computer.
2. Reinstall VPN client.
3. Verify latest VPN version is installed.

### Issue 3: Unable to Connect

Symptoms:

* Connection timeout
* Network unreachable

Resolution:

1. Check internet connectivity.
2. Restart router.
3. Switch network if possible.
4. Verify VPN server availability.

## Escalation Criteria

Escalate ticket if:

* Issue persists after reinstall.
* Multiple users report same issue.
* VPN server outage detected.
