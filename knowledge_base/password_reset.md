# Password Reset Standard Operating Procedure

## Purpose

Provide secure password reset procedures for employees.

## Password Requirements

* Minimum 12 characters
* One uppercase letter
* One lowercase letter
* One number
* One special character

## Reset Procedure

1. Visit Self-Service Password Portal.
2. Select "Forgot Password".
3. Complete MFA verification.
4. Create a new password.
5. Confirm password update.

## Common Issues

### MFA Failure

Resolution:

* Verify registered mobile device.
* Request MFA reset through IT.

### Account Locked

Resolution:

* Wait 15 minutes.
* Contact IT Support.

## Security Guidelines

Never share passwords with:

* Colleagues
* Vendors
* Managers
* Support staff
