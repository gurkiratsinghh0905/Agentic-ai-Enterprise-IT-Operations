# Remote Work Policy

## Purpose

Provide guidelines for secure remote work.

## Requirements

Employees must:

* Use company-managed devices.
* Connect using VPN.
* Enable MFA.
* Follow security policies.

## Prohibited Activities

* Using public computers
* Sharing work devices
* Storing company data on personal devices

## Security Requirements

* Device encryption mandatory
* Endpoint protection mandatory
* VPN mandatory

## Incident Reporting

Report lost devices immediately to IT Security.
