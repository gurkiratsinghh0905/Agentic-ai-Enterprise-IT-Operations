# Multi-Factor Authentication Setup

## Purpose

Protect enterprise accounts using MFA.

## Supported Methods

* Authenticator App
* SMS Verification
* Hardware Token

## Setup Process

1. Login to Employee Portal.
2. Open Security Settings.
3. Select Enable MFA.
4. Scan QR code using authenticator app.
5. Verify generated code.

## Common Issues

### Invalid Verification Code

Resolution:

* Sync device time.
* Generate a new code.

### Lost Device

Resolution:

* Contact IT Service Desk.
* Request MFA reset.
