# Device Onboarding Guide

## Purpose

Provision employee devices securely.

## Steps

1. Unbox company laptop.
2. Connect to internet.
3. Sign in using corporate credentials.
4. Complete MFA enrollment.
5. Install approved software.

## Mandatory Applications

* Outlook
* Teams
* VPN Client
* Endpoint Security Agent

## Verification Checklist

* Device encrypted
* Antivirus active
* VPN configured
* MFA enabled
