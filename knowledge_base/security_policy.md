# Enterprise Security Policy

## Security Principles

* Least Privilege Access
* Zero Trust Security
* MFA Enforcement
* Data Encryption

## Employee Responsibilities

Employees must:

* Protect credentials
* Lock devices when unattended
* Report suspicious activity
* Use approved software only

## Prohibited Activities

* Sharing passwords
* Installing unauthorized software
* Downloading unknown attachments
* Using personal storage services

## Incident Reporting

All security incidents must be reported within 30 minutes.
