# Outlook Email Configuration Guide

## Supported Versions

* Outlook 365
* Outlook 2021
* Outlook Web Access

## Setup Steps

1. Open Outlook.
2. Select Add Account.
3. Enter company email address.
4. Sign in using company credentials.
5. Complete MFA verification.

## Incoming Mail Server

mail.company.internal

## Outgoing Mail Server

smtp.company.internal

## Common Issues

### Emails Not Syncing

Resolution:

* Verify internet connectivity.
* Restart Outlook.
* Reauthenticate account.

### Authentication Errors

Resolution:

* Reset password.
* Verify MFA enrollment.
